# 变更日志

所有对项目的显著变更都会记录在此文件中。

格式基于 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/)，
并且本项目遵循 [语义化版本](https://semver.org/lang/zh-CN/)。

## [1.0.0] - 2023-11-24

### 新增
- 初始发布版本
- 支持识别.nfo文件及其关联视频
- 支持根目录和子目录中的残留文件处理
- 支持预览模式，安全检查待删除内容
- 支持回收模式，将删除内容移动到指定目录
- 支持直接删除模式，永久删除文件
- 跨平台支持：Windows和Linux/NAS系统
- Docker容器化部署支持
- 远程一键安装脚本
- 智能处理文件名冲突，避免覆盖
- 支持多种视频文件格式
- 支持多种残留文件类型识别

### 技术实现
- Python 3.x实现核心功能
- Windows批处理脚本
- Linux Shell脚本
- Docker容器支持
- GitHub Actions自动化构建（计划中）

### 文档
- 基本使用指南
- NAS设置指南
- 多平台支持说明
- Docker部署指南
- 贡献指南
